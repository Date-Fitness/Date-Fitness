## Hera

hera is a rest api web framework, and it is used like [pylon](git@github.com:xcodecraft/pylon-ng.git)  

## Installation
go get github.com/xcodecraft/hera

##Features
- RESTful route support
- HTTP Middleware
- Easy  YAML confg
-  Based syslog logkit
-  Redis pool
